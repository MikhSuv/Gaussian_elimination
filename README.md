# Метод Гаусса решения системы линейных уравнений

В корневой директории команда `make` создаст исполнимые файлы: `linear_slover` и `tests`.

## Основная программа `linear_slover`

Использование:
```bash
./linear_solver <gauss|jordan|choice>
```
Аргумент должен принимать одно из трех значений: `gauss`, `jordan` или `choice`
для решения основным методом Гаусса, методом Йордана и методом с выбором ведущего элемента соответственно.

Программа читает данные из файла `data.dat` и записывает решение в `result.dat`. На экран выводится модуль вектора невязки.

## Программа `tests`

Генерирует случайную систему порядка n=1000  и решает ее тремя методами. Выводит вектор невязки для каждого метода.
Создает матрицу Гильберта того же порядка и решает ее тремя методами. Выводит вектор невязки для каждого метода.
